# MD7 Sales Academy
## Design System

---

# Design Philosophy

The platform should feel premium.

Quiet.

Focused.

Executive.

Modern.

Avoid visual clutter.

Avoid excessive animation.

Motion should support understanding, never distract from it.

---

# Brand Colors

Primary Background

#052F47

Primary Text

White

Secondary Text

Soft Gray

Accent

Muted Teal

Success

Soft Green

Warning

Amber

Error

Muted Red

---

# Visual Language

Use Glassmorphism throughout.

Cards should appear layered.

Use:

Background blur

Semi-transparent panels

Soft borders

Large corner radius

Subtle shadows

Soft gradients

Minimal outlines

Avoid flat design.

---

# Typography

Use a modern sans-serif font.

Recommended hierarchy:

Display

Heading 1

Heading 2

Heading 3

Body

Caption

Code

Maintain generous spacing.

Readable line lengths.

Comfortable typography.

---

# Spacing System

Use an 8-point grid.

Consistent padding.

Consistent margins.

Consistent section spacing.

Whitespace should improve comprehension.

---

# Motion

Animations should feel natural.

Fade

Scale

Blur

Slide

Elevation

Hover

Focus

Loading Skeletons

Avoid bouncing.

Avoid flashy transitions.

---

# Cards

Cards become the primary UI element.

Cards contain:

Title

Summary

Content

Related Resources

Actions

Cards should always feel interactive.

---

# Buttons

Primary

Secondary

Ghost

Icon

Destructive

Buttons should have clear visual hierarchy.

---

# Progress Indicators

Progress should exist at:

Chapter Level

Competency Level

Course Level

Exercise Level

Overall Platform

---

# Icons

Use a consistent icon set.

Minimal.

Outline style preferred.

Avoid decorative icons.
